# 🏡 Apna Ghar Care – Property Management Website

Apna Ghar Care is a modern, responsive website designed for providing **home inspection, cleaning, and maintenance services** to homeowners living away from their property.

The platform focuses on **trust, convenience, and easy booking**, allowing users to connect instantly via WhatsApp and schedule appointments seamlessly.

---

## 🚀 Features

### ✅ WhatsApp Integration

* Floating WhatsApp button for instant contact
* Pre-filled message for quick inquiries
* Easy lead generation without login/signup

### 📅 Appointment Booking System

* Calendar-based date selection
* Time slot booking
* Auto redirect to WhatsApp with booking details

### 🔒 Trust-Building Sections

* Before/After service transformations
* Customer testimonials with photos
* “Why Choose Us” section

### 🎨 Modern UI/UX

* Clean and responsive design
* Mobile-friendly layout
* Smooth animations and hover effects

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Icons:** Font Awesome
* **Integration:** WhatsApp Click API
* **Hosting:** Netlify / GitHub Pages
